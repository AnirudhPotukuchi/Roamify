> **Discover Your Next Adventure.** Explore top destinations, taste exotic foods, and generate a transparent, customized travel budget.

**Roamify.** is a modern, responsive travel exploration and planning web application built as part of a Web Programming assignment. It provides users with an immersive onboarding experience to discover hand-picked destinations, learn about local culinary delights, and seamlessly plan an itinerary with real-time budget forecasting.

---

## ✨ Key Features

- **🎬 Immersive Video Onboarding:** A visually striking landing page with an auto-playing video background (`index.html`) leading user into the experience.
- **🔐 Secure Access Flow:** Intuitive login interface (`login.html`) connecting users to the main dashboard.
- **🗺️ Dynamic Travel Planner:** Personalize your trip parameters. A real-time engine computes living costs and travel expenses dynamically based on travelers, days, and selected destinations. 
- **🍲 Culinary Delights & Destinations:** Curated grids displaying popular cities and local street food favorites, complete with descriptions.
- **💳 Payment Success Flow:** A dedicated success confirmation step (`success.html`) immediately after finalizing trip details.
- **📱 Responsive UI/UX:** A sleek design implemented purely with Custom CSS (`styles.css`), integrating smooth transitions, hover effects, and Google's Poppins font for pristine typography across all devices.

---

## 🛠️ Technology Stack

Roamify is built utilizing core web technologies ensuring optimal performance without heavy framework dependencies.

* **HTML5:** Semantic architecture for layout across all application views (`HeroPage.html`, `login.html`, `success.html`, etc.).
* **CSS3:** Custom responsive styling utilizing CSS Flexbox, Grid, keyframe animations, and modern aesthetics.
* **JavaScript (Vanilla JS):** Core logic for dynamic DOM manipulation, trip cost calculation, and form validations.

## 🚀 Getting Started

1. **Clone the repository** to your local machine (or download the source directory).
2. Ensure you have the `Food/` folder and background assets (e.g. `Loading-Video.mp4`) in the project root.
3. Open `index.html` directly in your favorite modern browser (Chrome, Firefox, Safari, Edge).
4. Experience the introductory flow, proceed through the "Login" interface, and start adding destinations from the Dashboard!
